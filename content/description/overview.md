# Overview

## Overview

### Philosophy

Markdown is a simple way to format text that looks great on any device. You control the display of the document; formatting words as **bold** or _italic_, adding images, and creating lists are just a few of the things we can do with Markdown. Mostly, Markdown is just regular text with a few non-alphabetic characters thrown in, like `#` or `*`.
